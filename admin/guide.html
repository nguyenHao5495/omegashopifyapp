Age Verifier by OmegaTheme App Installation

Directly go to this link: https://apps.shopify.com/age-verifier-by-omega

After you install this app, That you finished setup and will be redirect to admin page to configure your app.

Note: images will were upload to asset of your store,so your store must allow for upload files

You only need put code <div class="ot-age-verifier"></div> anywhere you want to show your store: product page, collection page, cart page, home page,...

Step 1:
Go to Edit HMTL page of your theme: Online Store -> Themes -> Edit HTML/CSS.


Step 2:
Go to Edit cart page template: Templates -> theme.liquid


Step 3:
You can put this code before tag </body> on theme template: <div class="ot-age-verifier"></div>


App configure Tutorial

After installed Age Verifier by OmegaTheme App you will be redirect to admin page.

We have 2 main settings:

  • Layout setting: Configuration condition and popup layout such as: text, font size, text color ...
  • Image setting: Choose image background (if you use layout with background) and your store's logo

For using our app:

You only need put code <div class="ot-age-verifier"></div> anywhere you want to show your store: product page, collection page, cart page, home page,...

Uninstall Age Verifier App

If you don't want to use this app and want to uninstall it

You only need remove code <div class="ot-age-verifier"></div> where you put it in theme file


How can I change background image or logo image?

  1. Go to app setting
  2. Choos "Image Setting app"
  3. Browse to image to upload background or logo

Do I have to set error message?

You don't have to set that message. When visitors confirm wrong,the box dialog will shake to notice
